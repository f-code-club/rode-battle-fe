# Installation Guide

### Prerequisites

- [Bun](https://bun.sh/) version 1.3.6 or latest

### Setup

1. **Clone the repository:**

```bash
git clone https://github.com/f-code-club/rode-battle-fe.git
cd rode-battle-fe
```

2. **Install dependencies:**

```bash
bun install
```

3. **Run development server:**

```bash
bun run dev
```

### Build

```bash
bun run build
```
